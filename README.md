# Grok Promo Thread Builder

A simple, offline HTML tool to create, edit, save, and export X/Twitter threads — built specifically for promoting Grok.

![Grok Promo Thread Builder](Grok%20Promo%20Thread%20Builder.png)

## 🌐 Live Demo

Try it instantly here: **[https://pdragonlabs.github.io/grok-thread-builder](https://pdragonlabs.github.io/grok-thread-builder)**

## ✨ Features

- Live editable 4-part thread template
- Real-time character counting
- Save & Load threads as **JSON** (great for version control & reuse)
- One-click copy full thread
- Export as Markdown
- Quick launch buttons to:
  - X.com (for manual posting)
  - Buffer
  - Typefully
- Fully offline — no data sent anywhere
- Clean dark mode UI

## 🚀 How to Use

1. Open the **[Live Demo](https://pdragonlabs.github.io/grok-thread-builder)** or download the repo
2. Open `index.html` in any browser (works completely offline)
3. Edit the thread parts directly
4. Save as JSON to keep your drafts
5. Copy the full thread or export as Markdown
6. Launch your scheduler with one click
